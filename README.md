## node-chat-app
Implementation of a chat app using websockets in Node.js. The application makes use of the [socket.io](https://www.npmjs.com/package/socket.io) library.

### Quick start
#### Run server
    yarn start:server
    
#### Run client
    yarn start:client

#### Run both client and server
    yarn start
